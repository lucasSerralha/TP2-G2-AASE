Trabalho futuro:

Fase 6: Deployment

A fase final consiste em operacionalizar os modelos desenvolvidos e entregar valor ao negócio através de um sistema de BI.

Orquestração de Pipeline (ETL): Apache Airflow

Função: Será utilizado para construir e automatizar o pipeline de dados. Uma DAG (Directed Acyclic Graph) no Airflow irá orquestrar o processo de extrair os dados, executar os scripts de predição utilizando os modelos treinados e carregar os resultados na base de dados analítica.



Armazenamento (Base de Dados Analítica): PostgreSQL

Função: Servirá como o repositório central para os dados processados e para os resultados das predições geradas pelos modelos. O PostgreSQL é um sistema de gestão de base de dados relacional robusto e confiável, que servirá de fonte de dados para a ferramenta de BI.



Visualização (BI & Dashboards): Apache Superset

Função: Será a plataforma de Business Intelligence utilizada para construir os dashboards interativos. O Superset irá conectar-se ao PostgreSQL para consultar os dados e as previsões, apresentando-os de forma visual através de gráficos, tabelas e KPIs, permitindo assim a monitorização e a extração de insights pelos utilizadores finais
